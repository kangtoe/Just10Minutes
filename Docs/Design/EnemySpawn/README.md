**언제 참고:**
- 게임 후반 밸런싱 조정 시

## 📊 시스템 요약

```
시간 경과 → 예산 누적 → 주기적 스폰 체크
                ↓
         스폰 가능한 적 필터링 (시간 범위)
                ↓
         예산 내에서 적 선택 & 스폰
                ↓
         존재 점수 추적 → 동적 난이도 조정
```

**핵심 메커니즘:**
1. 시간대별 적 해금 (Light → Mid → Heavy)
2. 예산 기반 적 선택 (비용 시스템)
3. 동적 난이도 조정 (존재 점수 기반)
4. 시간에 따른 적 스탯 증가 (스케일링)
