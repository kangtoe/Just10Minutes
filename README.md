# Just 10 Minutes!

> **프로젝트명**: AutoSpaceShooter
> **게임 타이틀**: Just 10 Minutes!

Asteroids 조작 + Vampire Survivors 업그레이드 시스템을 결합한 10분 생존 보스 클리어 슈팅 게임

## 핵심 컨셉

**"간단한 조작, 전략적인 선택, 명확한 목표"**

자동으로 전진하는 우주선을 **회전만으로 조종**하며, 끊임없이 등장하는 적을 피하거나 격파하면서 **10분까지 생존하여 보스를 격파**하는 것이 목표입니다.

- **플레이 타임**: 10분+@
- **승리 조건**: 10분까지 생존하여 최종 보스 격파
- **간단한 조작**: 회전만으로 조종하는 원터치 플레이
- **모바일 최적화**: 짧은 세션, 명확한 목표

## 게임 플레이 루프

```
자동 전진 + 회전 제어 → 적 회피/격파 → 경험치 획득 → 레벨업
→ 업그레이드 선택 → 더 강한 적 등장 → 10분 생존 → 보스 격파 → 승리
```

## 조작 방식

- **자동 전진**: 기체가 바라보는 방향으로 자동 이동
- **자동 사격**: 기체가 자동으로 전방 사격
- **회전 조작**: 화면 좌/우 절반을 터치/클릭하여 기체 회전
- **원터치 플레이**: 회전만 제어하면 되므로 모바일 최적화

## 게임 규칙

### 승리/패배 조건
- **승리**: 10분까지 생존하여 출현한 보스를 격파
- **패배**:
  - 내구도 0 (실드 소진 후)
  - 화면 이탈 (경계 벗어나면 즉사)

### 생존 시스템
- **실드**: 모든 피해를 우선 흡수, 시간 경과 시 자동 재생
- **내구도**: 실드 소진 후 마지막 방어선, 재생 어려움

### 공격 방식
- **자동 사격**: 발사체로 원거리 공격 (멀티샷, 관통, 데미지 업그레이드 가능)
- **충돌 공격**: 적과 부딪혀 피해를 가함 (충돌 피해 업그레이드 가능)

## 핵심 시스템

### 시간 기반 스폰 시스템 (10분 구조)
- **연속 스폰**: 일정 간격마다 가중치 기반으로 적 선택 및 스폰
- **난이도 상승**: 시간 경과에 따라 적 스탯 증가 & 스폰 간격 감소
- **적 해금**: 0-3분(소형) → 3-7분(중형) → 7-10분(대형) → 10분(최종 보스)

### 업그레이드 시스템
레벨업 시 3가지 업그레이드 중 1개 선택으로 다양한 빌드 구성
- 최대 내구도/실드, 실드 재생
- 멀티샷, 충돌 피해
- 연사 속도, 발사체 데미지/속도 (Phase 2 예정)
- 이동/회전 속도, 관통, 넉백 등 (Phase 2 예정)

### 적 시스템
14종의 적 유형 (Light 4종, Mid 7종, Heavy 2종, Boss 1종)

## 기술 스택

- **엔진**: Unity 2D
- **입력**: Unity New Input System (키보드/마우스/게임패드/모바일)
- **물리**: Rigidbody2D (AddForce 기반)
- **패턴**: Singleton Manager, Object Pooling (구현 예정)

## 게임 레퍼런스

- **Vampire Survivors**: 시간 기반 스폰, 레벨업/업그레이드 시스템
- **Nova Drift**: 빌드 시너지, 업그레이드 깊이
- **Asteroids**: 회전 + 전진 메커닉
